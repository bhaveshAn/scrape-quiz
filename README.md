# LegalStart Tech Case - Web Scraper

## Steps to Install and Run on Linux

### For Python 2

```
virtualenv -p python2 venv
source venv/bin/activate
pip install -r requirements.txt
python scraper.py

```

### For Python 3

```
virtualenv -p python3 venv
source venv/bin/activate
pip install -r requirements.txt
python3 scraper.py

```
